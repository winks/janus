## Janus

This is a small parser that allows the following JavaScript expressions:

  * `expr == expr`
  * `expr != expr`
  * `expr >= expr`
  * `expr > expr`
  * `expr <= expr`
  * `expr < expr`
  * `expr || expr`
  * `expr && expr`
  * `( expr )`
  * `variable`
  * `3`
  * `-4`
  * `'"There", he said.'`
  * `"\"There\", he said."`
